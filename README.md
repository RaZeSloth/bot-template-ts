# Example ts bot template
Make more commands like so
```bath
commands
    ├── test
    │   └── test.ts 
    └── util
         └── somefile.ts
```
# Env
Example of the env
```bath
token=YourToken
prefix=!
```

# Starting the project

First, you'd have to build the project with `yarn install` to start the bot run `yarn run start`
